TRON-FundYou

A decentralized investment platform linking investors and creators without intermediaries on the TRON blockchain. Developed during TRON Builder Tour NYC, it provides secure and verifiable transactions using smart contracts.

Technologies Used: TRON Blockchain, Smart Contracts, JavaScript, HTML/CSS, React

Features:

  • Decentralized connection between investors and project creators.
  • Smart contracts for secure and verifiable transactions.
  • React UI for live TRON (TRX, USDT) price updates and wallet authentication using TRONLink.
  • Successful project funding of ~10,000 TRX during the hackathon.

Inspiration: To create a platform that simplifies connecting investors with project creators through secure, decentralized transactions.

Challenges Faced: Ensuring secure smart contract implementation and handling live updates for TRON prices.

Future Plans: Adding features for a decentralized voting system for funding decisions.
