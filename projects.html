Meal Mentor

A meal-tracking iOS app with AI-driven nutritional advice built using Swift/SwiftUI and OpenAI API.

Technologies Used: Swift, SwiftUI, Python, SQLite, OpenAI API

Features:

  • Daily dietary logging with a unique entry system.
  • AI-driven nutritional advice and meal suggestions.
  • Backend for database connectivity storing user entries for personalized insights.
  • Chat interface for real-time AI interactions for dietary guidance and macro balancing.

Inspiration: Daily fitness and activity trends inspired us to hold ourselves accountable for our diet. The app allows tracking of calories, macros, and more through entries.

Challenges: Integration of APIs and ensuring encryption was difficult, and the app is currently functional only locally.

Future Plans: API integration improvements and adding a centralized database.
